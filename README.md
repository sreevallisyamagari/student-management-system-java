# Student Management System

A simple Java console-based application developed using Object-Oriented Programming (OOP) concepts.

## Features

* Add Student
* View All Students
* Search Student by Roll Number
* Update Student Details
* Delete Student
* Menu-Driven Interface

## Technologies Used

* Java
* OOP Concepts
* ArrayList
* Scanner Class

## Concepts Covered

* Classes and Objects
* Constructors
* Encapsulation
* ArrayList
* Loops
* Conditional Statements
* CRUD Operations

## Project Structure

Main.java

## How to Run

1. Compile the program:

javac Main.java

2. Run the program:

java Main
